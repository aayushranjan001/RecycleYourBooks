# RecycleYourBooks

An ecommerce android app which lets user buy and sell books. 
The user needs to login to the app using either their google account or through the phone number. When logged in the user can put the details of the books which is stored in the Firebase database. The buyer can browse through the catalog of books available. Each book details contain the user details as well such as the contact number and also shares the location of the seller. The buyer can contact the seller directly and can buy the book.
